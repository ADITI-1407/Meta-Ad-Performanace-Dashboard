# Meta-Ad-Performanace-Dashboard
🚀 Power BI Project: Meta Ad Performance Analysis
Here’s how I analyzed and derived insights from Meta Ads data 👇
🔹 Overview
• Built an interactive dashboard to track campaign performance
• Covered full funnel: Impressions → Clicks → Engagement → Conversions → Purchases
• Focused on KPIs like CTR, engagement rate, conversion rate, and budget efficiency
🔹 Funnel Insights
• 216K impressions & 25.4K clicks → CTR: 11.76% (very high)
• Engagement rate: 13.56% (strong user interaction)
• Only 1.3K purchases → Conversion rate: 5.21% (drop observed)
• Purchase rate from impressions: 0.61%
👉 Insight: Strong awareness & engagement, but weak conversion → funnel leakage
🔹 Audience Analysis
• Females (43%) engage more than males (22%)
• Age group 18–30 drives maximum interaction
• Key markets: India & Brazil (high volume)
• High-value markets: Germany & UK
👉 Insight: Young female audience is core target → needs focused strategy
🔹 Time-Based Insights
• Peak engagement during afternoon & evening
• Certain dates (19–21, 25–27) show spikes
👉 Insight: Event-based campaigns & time targeting improve performance
🔹 Ad Format Performance
• 🎥 Video Ads → Best CTR, engagement & conversions
• 📱 Story Ads → Strong performance
• 🖼️ Image & Carousel → Lower conversion efficiency
👉 Insight: Shift budget towards Video & Stories
🔹 Key Recommendations
• Optimize landing pages to improve conversions
• Use retargeting for engaged but non-converting users
• Focus on high-performing audience segments
• Allocate budget to video/story formats
• Schedule ads during peak engagement hours
• Use different strategies for high-volume vs high-value regions
💡 Conclusion:
Campaigns are great at grabbing attention but need optimization in the conversion stage to maximize ROI.
